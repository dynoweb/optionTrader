This project is based on EclipseLink JPA

To create a JPA Model Class, right click on the project and select New -> JPA Entities from Table 

To set up the project in Eclipse, a user Library named myJPA2.  Include the following jars

eclipselink/jlib/eclipselink.jar
eclipselink/jlib/eclipselink-src.jar
eclipselink/jlib/jpa/javax.persistence.source_2.1.0.v201304.....jar
eclipselink/jlib/jpa/javax.persistence._2.1.0.v201304.....jar
eclipselink/jlib/jpa/org.eclipse.persistence.jpa.modelgen.source_2.5.0.v2015...jar
eclipselink/jlib/jpa/org.eclipse.persistence.jpa.modelgen_2.5.0.v2015...jar
